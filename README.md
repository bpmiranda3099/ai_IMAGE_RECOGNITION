## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/my-nodejs-project.git
   ```
2. Navigate into the project directory:
   ```
   cd my-nodejs-project
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To start the application, run:
```
node src/index.js
```
